# MonsterShop

This project has been converted to a React + Vite app using the `Panel de control.html` design.

## Fast start

- `npm install`
- `npm run dev`

The live app entrypoint is `public/index.html` and the React root component is `src/AppRoot.jsx`.

## Notes

- The active React dashboard is built from `src/AppRoot.jsx` and modular components under `src/components/`.
- Data and design constants are stored in `src/data/dashboardData.js`.
- Legacy static HTML/CSS files are no longer used by the React app.
