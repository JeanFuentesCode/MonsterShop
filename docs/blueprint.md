# **App Name**: ComandaFlow

## Core Features:

- Gestión Modular de Productos: Sistema para administrar el catálogo de artículos con categorías dinámicas, permitiendo adaptar la app a cualquier negocio más allá de las tortas.
- Seguimiento de Pedidos y Cobros: Módulo diseñado para rastrear entregas y pagos pendientes, permitiendo marcar pedidos como 'entregados' y posteriormente 'cobrados' para gestionar ventas a crédito.
- Persistencia en Tiempo Real con Firebase: Sincronización instantánea de datos entre dispositivos móviles y escritorio utilizando Firebase Firestore como base de datos centralizada.
- Control de Stock Automatizado: Actualización automática de niveles de inventario tras cada pedido completado, con alertas visuales de bajo stock.
- Generador de Catálogo IA: Una herramienta que utiliza IA para sugerir descripciones optimizadas y nombres creativos para los productos basándose en ingredientes o etiquetas.
- Dashboard de Análisis de Ventas: Panel visual con métricas clave sobre ingresos, productos más vendidos y balance de deudas pendientes.
- Sistema de Temas Dinámicos: Interfaz con soporte nativo para temas claro y oscuro, optimizado para visualización en pantallas Retina de iPhone y monitores de PC.

## Style Guidelines:

- Color primario: Verde Bosque (#51A329) para botones y elementos de acción principales.
- Acento: Lima Neón (#CEEB47) para resaltar estados importantes y notificaciones.
- Fondos: Blanco verdoso suave (#F1F5F0) en tema claro y Gris Carbón Profundo (#1A1C19) en tema oscuro.
- Fuente principal: 'Inter' para una legibilidad superior en interfaces de gestión de datos y dispositivos móviles.
- Iconos de línea minimalista (estilo Lucide o Phosphor) para mantener un diseño limpio y moderno.
- Diseño 'Mobile-First' basado en CSS Grid y Flexbox, garantizando fluidez total sin depender de frameworks pesados.
- Transiciones suaves de 200ms para cambios de estado y apertura de modales, mejorando la sensación de modernidad.
